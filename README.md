This project is a dynamic web application that integrates an AI API for advanced functionality. It features a responsive design using Bootstrap, while React powers the interactive front-end. The application utilizes JavaScript for smooth user interactions and data handling. The back-end API manages AI requests, delivering fast, real-time responses.

Live Here [  https://react-ai-writing.vercel.app/ ]

Tech Stack:
Frontend: React, Bootstrap, JavaScript
Backend: OpenRouterAI API
Others: HTML, CSS
This stack provides a responsive, scalable, and user-friendly interface with AI-powered capabilities.
